# AI-Assisted Task Management API

Minimal TypeScript Express REST API layered into data, logic, and presentation layers.

Quick start

1. Copy `.env.example` to `.env` and set `DATABASE_URL`.
2. Install deps: `npm install`.
3. Run Prisma migrate: `npm run prisma:migrate` (creates sqlite dev.db by default).
4. Start dev server: `npm run dev`.
