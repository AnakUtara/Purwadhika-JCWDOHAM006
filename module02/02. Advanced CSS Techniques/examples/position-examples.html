CSS Position Property Examples

position: static (default)

This box is static (default position).

position: relative

This box is relative (moved 30px right, 10px down).

position: absolute (inside relative container)

Explanation: When an element is position: absolute inside a position: relative container, it is positioned relative to that container.

This box is absolute (top: 10px, left: 10px inside relative container).

position: absolute (no relative parent)

Explanation: When an element is position: absolute and there is no relative parent, it is positioned relative to the <body> (the page itself).

This box is absolute with no relative parent (top: 40px, right: 400px on the page).

position: fixed

This box is fixed (top: 10px, right: 10px on screen).

position: sticky

Explanation: The sticky box will "stick" to the top of its scrollable container when you scroll down.

Scroll down to see sticky effect
This box is sticky (sticks to top when scrolling).
